Classes

Classes are everything put together.
Functions inside a class are called "methods"
Variables inside a class are called "properties"
If statements, for loops, arrays, objects are all allowed inside a class
Classes can be extended, so you can add your own functionality to them



...

Example class myAnimal { constructor(passName, passAge) { this.myName = passName; this.myAge = passAge; } myShow(){ document.getElementById('myDiv130').innerHTML = 'Hello '+ this.myName + '. You are ' + this.myAge + ' years old
' } } // end class myAnimal myDog = new myAnimal('Snickers', 5) myDog.myShow()







Extend class

Here we extend the class myAnimal to myCat



...

Example class myAnimal { constructor(passName, passAge) { this.myName = passName; this.myAge = passAge; } myShow(){ document.getElementById('myDiv131').innerHTML = 'Hello '+ this.myName + '. You are ' + this.myAge + ' years old
' } } // end class myAnimal class myCat extends myAnimal { myShow(){ document.getElementById('myDiv131').innerHTML = this.myName + ' is a great cat who is ' + this.myAge + ' years old
' } } // end extended class myCat myPet = new myCat('Fluffy', 14) myPet.myShow()
