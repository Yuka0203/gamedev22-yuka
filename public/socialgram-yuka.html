Summarize: what Yuka did learn in Game Dev

    The Nine components of programming

  • Variables
    -store stuff
    x = 3
    myname = 'Yuka'


  • Input / Output
    - information into and out of your program
    x = document.getElementById('myText01').value // example input
    document.getElementById('myDiv01').innerHTML = x // example output


  • Decisions
    -decide stuff based on stored stuff
    if(x==3) {}
    if (x <= 3){} else {}
    if ((x > 3) && (x < 9)) {} else {}


  • Loops
    -repeat stuff
    for (x=0; x < 3; x++){ // example for loop
    }


  • Events
    -trigger computer code
    onClick(), onMouseOver(), onBlur(), onFocus() abouit 60 learn them


  • Arrays
    -a numbered list of variables
    x[0] = 3;
    x[1] = 6;


  • Functions
    - named group of code that can be called multiple times
    function myFun() { } // defines a group of code
    myFun() // activates the function


  • Objects
    - a named list of variables
    x = {age:20, name:'fred'} // sets object
    // uses object
    x.age
    x.name


  • Classes
    - groups variables and functions to make programming easier
    methods --> functions
    properties --> variable
